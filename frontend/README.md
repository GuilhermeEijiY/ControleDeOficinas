#  Frontend - Sistema de Controle de Oficinas

> Interface React.js para gerenciamento de oficinas do projeto ELLP

##  Sobre
Frontend desenvolvido em React.js com interface moderna para controle de oficinas, usuários e inscrições.

##  Tecnologias
- **React 19.2.0** - Biblioteca principal
- **React Router DOM** - Navegação entre páginas  
- **CSS3** - Estilização e design responsivo
- **JavaScript ES6+** - Lógica da aplicação

##  Instalação
```bash
# Clone o repositório
git clone https://github.com/GuilhermeEijiY/ControleDeOficinas.git

# Entre na pasta do frontend
cd ControleDeOficinas/frontend

# Instale as dependências
npm install

# Execute o projeto
npm start
